# Description
replace the annoyingly slow tank death animation with an instant-ragdoll death. this will also remove the collision that persists as part of the death animation.

## soft requirements
- [hxstocks](../hxlib/scripting/include/hxstocks.inc) - to compile

# Usage
there's a convar to toggle the plugin on/off:
- `skip_tank_death_animation` 1 or 0
