# Description
per-client configuration of the sv_rollangle convar.

## Soft Requirements
- [cookie_manager](../cookie_manager/README.md) - to apply cookie values instantly, and register itself for a menu

# Usage
Server operators can choose the default value of the cookie with the convar:
`cookie_roll_angle`

clients can customize the value of their personal roll angle setting with the cookie:
`roll_angle`
