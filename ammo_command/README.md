# Description
adds command players can use to check reserve ammo.

this is useful if your server sets the reserve ammo of any weapon beyond 1000, since that triggers the abnormal ammo display on a client's HUD (stuck at 999)

# Usage
## Command
- `/ammo` - give no args to see usage of your equipped weapon's reserve ammo. for a specific weapon slot, give it as a number 1 through 5 as the argument.
