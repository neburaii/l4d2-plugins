# Description
this plugin ensures weapons always spawn with full ammo.

if players finish a map, they will normally transition to the next map with the same missing ammo as they ended with. this plugin will make it so they refill to max in these scenarios.

# Usage
## ConVar
- `autofill_ammo_on_spawn` - 1 to enable plugin, 0 to disable
